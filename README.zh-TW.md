這是一套給 Codex 用的遊戲工作室外殼，重點是路由、分波次、監工、收尾。

## 先看這裡

- 你已經知道要做什麼，就直接丟需求給 AI。
- 你不知道要用哪個指令，就讓 Codex 自己判斷下一條安全流程。
- 想完整走工作室流程，就從 `/start` 開始。

## 安裝

1. 打開這個 repo：`https://github.com/Gale0418/Codex-Game-Studios`
2. 把它安裝成 Codex skill
3. 如果你沒有 installer，就把整包放到 `~/.codex/skills/codex-game-studios/`
4. 重啟 Codex，讓它重新掃 skills
5. 新 session 先跑 `/start`

## 呼叫方式

- `請用 codex-game-studios skill 幫我看這個專案`
- `用工作室流程 review 這個遊戲`
- `直接幫我判斷這個任務要不要開多 agent`
- `請讓 AI 自行判斷下一個安全流程`

## 備註

- 這份是從 Claude Game Studio 的概念與結構改造而來，再整理成 Codex 可直接使用的版本。
- 如果不知道怎麼用，就直接把需求丟給 AI，讓它自己判斷 lane。
- 最短安全入口請看 `SKILL.md`、`AGENTS.md`、`references/codex-first.md`
