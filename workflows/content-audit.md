# Content Audit

## Purpose

- Review content for gaps, contradictions, and stale references.

## Trigger

- Use when docs, narrative, or content sets need a consistency pass.

## Primary lane

- `narrative-director`
- `writer`
- `docs-release`

## Inputs

- Content set
- Source of truth
- Target audience

## Steps

1. Identify what content is in scope.
2. Find missing, stale, or conflicting material.
3. Call out the highest-value corrections.
4. Hand back the audit list.

## Outputs

- Audit scope
- Gaps
- Action items

## Exit criteria

- The content set can be corrected or accepted.

## Template

- `templates/content-audit.md`
