# Art Bible

## Purpose

- Define the visual style targets and asset boundaries for the project.

## Trigger

- Use when art direction or asset style needs to be clarified.

## Primary lane

- `art-director`
- `technical-artist`
- `ux-designer`

## Inputs

- Visual targets
- Asset references
- Constraints

## Steps

1. State the intended art direction.
2. Define the reusable asset rules.
3. Call out what must stay consistent.
4. Hand back the art bible notes.

## Outputs

- Style targets
- Asset rules
- Notes

## Exit criteria

- The visual direction is clear enough to guide production.

## Template

- `templates/art-bible.md`
