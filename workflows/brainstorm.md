# Brainstorm

## Purpose

- Generate candidate directions before committing to a design path.

## Trigger

- Use when the solution space is still open.

## Primary lane

- `game-designer`
- `creative-director`
- `systems-designer`

## Inputs

- Goal
- Constraints
- Similar patterns

## Steps

1. Generate a few viable ideas.
2. Compare tradeoffs and player impact.
3. Cut weak options quickly.
4. Name the strongest direction.

## Outputs

- Ideas
- Constraints
- Candidate direction

## Exit criteria

- The next design pass has a preferred direction.

## Template

- `templates/brainstorm-notes.md`
