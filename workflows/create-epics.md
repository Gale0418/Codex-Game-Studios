# Create Epics

## Purpose

- Turn a broad scope into ordered epics.

## Trigger

- Use when a large body of work needs a higher-level breakdown.

## Primary lane

- `producer`
- `architecture`
- `game-designer`

## Inputs

- Scope
- Deadlines
- Constraints

## Steps

1. Group the scope into epics.
2. Define dependencies and order.
3. Call out what is blocked or parallel.
4. Hand back the epic list.

## Outputs

- Epics
- Dependencies
- Sequence

## Exit criteria

- The work can be split into story-level planning.

## Template

- `templates/create-epics-plan.md`
