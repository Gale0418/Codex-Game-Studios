# Production State Schema

## Current state fields

- `owner`
- `current_focus`
- `active_lanes`
- `current_stage`
- `current_gate`
- `last_verified`
- `open_risks`
- `next_gate`

## Active note shape

- Keep `production/active.md` aligned to the fields above.
- Update the active note when the stage or lane changes.
- Keep the note short enough to scan quickly.

## History shape

- Record the date, the new milestone, and the main reason for the change.
- Keep one entry per meaningful checkpoint.
- Do not turn history into a transcript.
